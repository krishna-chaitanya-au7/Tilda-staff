# Project Brief

## Project Goals
Update Tilda-staff mobile application to align messaging features with bissfest_tool web application and add profile management.

## Core Requirements
1.  **Messages Screen Updates**:
    -   Resize "New Conversation" card (height 50%).
    -   Add "Participants" card below it (functional).
    -   Add Poll creation and Image upload to message input.
    -   Update message bubbles with date/time and read receipts.
2.  **Navigation**:
    -   Add "Profile" tab to bottom bar.
3.  **Profile Screen**:
    -   Display user details.
    -   Edit functionality.
    -   Logout functionality.
